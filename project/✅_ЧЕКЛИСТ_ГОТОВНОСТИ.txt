❓ Что такое Django?
   → Веб-фреймворк на Python для быстрой разработки

❓ Что такое MVT?
   → Model (данные), View (логика), Template (отображение)

❓ Как работает система отзывов?
   → Пользователь → Форма → Проверка → БД → Модерация → Публикация

❓ Зачем нужна модерация?
   → Защита от спама и неприемлемого контента

❓ Что такое миграции?
   → Инструкции для изменения структуры базы данных

❓ Что делает {% csrf_token %}?
   → Защищает от CSRF атак (подделки запросов)

❓ Что такое ORM?
   → Работа с БД через Python объекты вместо SQL

❓ Какая база данных используется?
   → SQLite (легкая файловая БД)
