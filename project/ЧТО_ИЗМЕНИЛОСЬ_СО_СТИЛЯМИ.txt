Если спросят про стили, объясни:

❓ Почему стили в отдельном файле?
✅ Это правильная практика веб-разработки
✅ Разделение контента (HTML) и оформления (CSS)
✅ Стили кэшируются браузером - быстрее загрузка
✅ Легче поддерживать и изменять

❓ Что такое {% static %}?
✅ Это тег Django для работы со статическими файлами
✅ Автоматически генерирует правильный путь к файлу
✅ Работает и в разработке, и в production

❓ Где хранятся статические файлы?
✅ В папке static/ в корне проекта
✅ CSS файлы в static/css/
✅ JavaScript в static/js/
✅ Изображения в static/images/

❓ Что такое {% load static %}?
✅ Подключает систему статических файлов Django
✅ Нужно добавлять в начало каждого шаблона
✅ После этого можно использовать {% static %}
