Знай эти 3 вещи:

1. MVT паттерн:
   Model (данные) → View (логика) → Template (отображение)

2. Как работает система отзывов:
   Форма → Проверка → БД → Модерация → Публикация

3. Модель Review:
   author_name, email, rating, text, created_at, is_approved

Основные файлы системы отзывов:

game/models.py          - Модель Review (структура данных)
game/views.py           - Функция reviews() (логика)
game/urls.py            - Маршрут /reviews/ (URL)
game/forms.py           - Форма ReviewForm (валидация)
game/admin.py           - Админ-панель (управление)
templates/game/reviews.html  - Страница отзывов (HTML)

Вспомогательные файлы:

add_test_reviews.py     - Скрипт для добавления тестовых отзывов
restore_data.py         - Скрипт для восстановления данных
